# Marketing assets

Social/promo assets for DevRadar. Brand is ported from `reference/devradar-site.html`
(gold `#E2B65B` on near-black `#060607`), never redesigned.

## X / Twitter banner

- `devradar-x-banner.png` — 1600×900 (16:9, X in-stream). Post-ready.
- `devradar-x-banner.svg` — source vector (regenerated by the build script).
- `fonts/` — Geist Sans + Geist Mono (OFL) used for rendering. Geist stands in
  for the site's Clash Display / Switzer, which aren't open-download.

Brand mark is the gold eye logo (vector, redrawn so it scales crisply). Socials:
X [@DevRadarS](https://x.com/DevRadarS) · Telegram [t.me/devradars](https://t.me/devradars).

### Regenerate

```bash
cd marketing
npm i @resvg/resvg-js   # if not already installed
node build-banner.js    # writes devradar-x-banner.{svg,png}
```
